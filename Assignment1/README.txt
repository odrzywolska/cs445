CS445 Assignment 1

The purpose of this project was to learn inheritance and polymorphism through the use of Thing and Creature classes.

Note: Java and javac have to be defined in the path
      The junit4.jar must be in the default location (/usr/share/java/junit4.jar)

To compile project:
   $ make

To execute project:
   $ make run

To execute JUnit tests:
   $ make JUnitTest

To clean project:
   $ make clean
